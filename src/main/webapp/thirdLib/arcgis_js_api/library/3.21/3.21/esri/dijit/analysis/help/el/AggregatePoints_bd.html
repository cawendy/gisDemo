Συνάθροιση Σημείων

Το εργαλείο αυτό λειτουργεί με ένα σημειακό θεματικό επίπεδο και ένα θεματικό επίπεδο περιοχής. Τα στοιχεία περιοχής εισόδου μπορεί να προέρχονται από ένα πολυγωνικό θεματικό επίπεδο ή μπορεί να είναι τετράγωνα ή εξάγωνα bin που υπολογίζονται κατά την εκτέλεση του εργαλείου. Το εργαλείο αρχικά προσδιορίζει τα σημεία που ανήκουν σε κάθε καθορισμένη περιοχή. Αφού προσδιοριστεί αυτή η χωρική σχέση σημείου και περιοχής, υπολογίζονται τα στατιστικά στοιχεία για όλα τα σημεία της περιοχής και αντιστοιχίζονται στην περιοχή. Το πιο βασικό στατιστικό στοιχείο είναι η μέτρηση του αριθμού των σημείων εντός της περιοχής, αλλά μπορείτε, επίσης, να υπολογίσετε και άλλα στατιστικά στοιχεία.

Για παράδειγμα, ας υποθέσουμε ότι έχετε στοιχεία σημείου για τοποθεσίες με καφετέριες και στοιχεία περιοχής για επαρχίες και θέλετε να συνοψίσετε τις πωλήσεις καφέ ανά επαρχία. Θεωρώντας ότι οι καφετέριες διαθέτουν ένα περιγραφικό γνώρισμα TOTAL_SALES, μπορείτε να βρείτε το άθροισμα όλων των TOTAL_SALES κάθε επαρχίας, τις μέγιστες ή ελάχιστες τιμές TOTAL_SALES κάθε επαρχίας, ή άλλα στατιστικά στοιχεία, όπως το πλήθος, την τυπική απόκλιση και την διακύμανση.

Το εργαλείο μπορεί επίσης να λειτουργεί με δεδομένα που έχουν χρονική πληροφορία. Αν είναι ενεργοποιημένος ο χρόνος στα σημεία εισόδου, τότε είναι διαθέσιμες οι επιλογές χρονικής κατάτμησης. Η χρονική κατάτμηση σάς επιτρέπει να υπολογίσετε τη σχέση σημείου και περιοχής, εξετάζοντας ένα συγκεκριμένο χρονικό τμήμα. Για παράδειγμα, θα μπορούσατε να εξετάσετε ωριαία διαστήματα, πράγμα που θα είχε ως αποτέλεσμα εξόδους για κάθε ώρα.

Για ένα παράδειγμα που αφορά το χρόνο, ας υποθέσουμε ότι είχατε σημειακά στοιχεία κάθε συναλλαγής που έγινε σε διάφορες τοποθεσίες με καφετέριες και κανένα πολυγωνικό θεματικό επίπεδο. Τα δεδομένα έχουν καταγραφεί για πάνω από ένα χρόνο και κάθε συναλλαγή έχει μια τοποθεσία και χρονική σήμανση. Θεωρώντας ότι κάθε συναλλαγή έχει ένα περιγραφικό γνώρισμα TOTAL_SALES, μπορείτε να βρείτε το άθροισμα όλων των TOTAL_SALES εντός του χώρου και του χρόνου ενδιαφέροντος. Αν αυτές οι συναλλαγές αφορούν μία μόνο πόλη, θα μπορούσαμε να δημιουργήσουμε περιοχές που αποτελούν πλέγματα 1 χιλιομέτρου και να εξετάσουμε εβδομαδιαία χρονικά τμήματα ώστε να συνοψίσουμε τις συναλλαγές τόσο χρονικά όσο και χωρικά.

Επιλέξτε ένα θεματικό επίπεδο που να περιέχει σημεία τα οποία θα συναθροιστούν σε περιοχές

Το σημειακό θεματικό επίπεδο για το οποίο θα γίνει συνάθροιση στις περιοχές. Για τα σημειακά θεματικά επίπεδα που συναθροίζονται σε τετράγωνα ή εξάγωνα bin, η χωρική αναφορά της επεξεργασίας πρέπει να είναι ορισμένη σε προβολικό σύστημα συντεταγμένων μέσα από τα Περιβάλλοντα ανάλυσης.

Εκτός από την επιλογή ενός θεματικού επιπέδου από το χάρτη σας, μπορείτε επίσης να επιλέξετε Αναζήτηση θεματικών επιπέδων στο κάτω μέρος της αναπτυσσόμενης λίστας για αναζήτηση στα περιεχόμενά σας ενός κοινόχρηστου αρχείου συνόλου δεδομένων με big data ή ενός feature layer.

Διαλέξτε ένα θεματικό επίπεδο περιοχής για συνάθροιση σε

Οι περιοχές στις οποίες θα συναθροιστεί το σημειακό θεματικό επίπεδο. Μπορείτε να επιλέξετε είτε ένα θεματικό επίπεδο περιοχής είτε bin τα οποία θα δημιουργηθούν κατά την εκτέλεση του εργαλείου. Κατά τη δημιουργία των bin, για Τετράγωνο, ο αριθμός και οι μονάδες που καθορίζονται προσδιορίζουν το ύψος και το μήκος του τετραγώνου. Για Εξάγωνο, ο αριθμός και οι μονάδες που καθορίζονται προσδιορίζουν την απόσταση μεταξύ των παράλληλων πλευρών.

Για την ανάλυση με χρήση τετράγωνων ή εξάγωνων bin, απαιτείται προβολικό σύστημα συντεταγμένων. Μπορείτε να ρυθμίσετε το Σύστημα συντεταγμένων επεξεργασίας στα Περιβάλλοντα ανάλυσης. Εάν το σύστημα συντεταγμένων επεξεργασίας δεν έχει οριστεί σε προβολικό σύστημα συντεταγμένων, θα σας ζητηθεί να το ορίσετε κατά την Εκτέλεση ανάλυσης.

Διαλέξτε το πολυγωνικό θεματικό επίπεδο για συνάθροιση

Το πολυγωνικό θεματικό επίπεδο όπου θα συναθροιστούν τα σημεία. Τα σημεία που βρίσκονται εντός των ορίων των περιοχών αυτού του επιπέδου θα μετρηθούν και, προαιρετικά, θα συνοψιστούν με την επιλογή στατιστικής που κάνατε.

Πρέπει να οριστεί είτε το Μέγεθος bin για bin είτε ένα Θεματικό επίπεδο περιοχής.

Διαλέξτε απόσταση για να δημιουργήσετε bin και να κάνετε συνάθροιση σε

Η απόσταση που χρησιμοποιείται για τη δημιουργία bin στα οποία θα συναθροιστούν τα σημεία εισόδου. Για ένα τετράγωνο bin, το μέγεθος αντιπροσωπεύει το ύψος του τετραγώνου. Για εξάγωνο, το μέγεθος αντιπροσωπεύει το ύψος του εξαγώνου (από τις παράλληλες πλευρές).

Πρέπει να οριστεί είτε το Μέγεθος bin για bin είτε ένα Θεματικό επίπεδο περιοχής.

Συνάθροιση χρησιμοποιώντας χρονικά βήματα (προαιρετικό)

Αν είναι ενεργοποιημένος ο χρόνος στο σημειακό θεματικό επίπεδο εισόδου, και ως τύπος έχει επιλεχθεί στιγμή, η ανάλυση μπορεί να γίνει χρησιμοποιώντας χρονική κατάτμηση. Υπάρχουν τρεις παράμετροι που μπορείτε να ορίσετε με χρήση του χρόνου:

  • Διάστημα χρονικού βήματος
  • Συχνότητα επανάληψης χρονικού βήματος
  • Χρόνος για ευθυγράμμιση των χρονικών βημάτων

Φανταστείτε ότι έχετε δεδομένα που αναπαριστούν χρονικά ένα έτος και θέλετε να κάνετε ανάλυση χρησιμοποιώντας εβδομαδιαία χρονικά βήματα. Σε αυτή την περίπτωση, πρέπει να ορίσετε το Διάστημα του χρονικού βήματος σε 1 εβδομάδα.

Φανταστείτε ότι έχετε δεδομένα που αναπαριστούν χρονικά ένα έτος και θέλετε να κάνετε ανάλυση χρησιμοποιώντας την πρώτη εβδομάδα του μήνα. Σε αυτή την περίπτωση, πρέπει να ορίσετε το Διάστημα χρονικού βήματος σε 1 εβδομάδα, τη Συχνότητα επανάληψης χρονικού βήματος σε 1 μήνα, και τον Χρόνο για ευθυγράμμιση των χρονικών βημάτων σε 1 Ιανουαρίου, 12:00 πμ.

Διάστημα χρονικού βήματος

Χρονικό διάστημα για τη δημιουργία των χρονικών βημάτων. Το διάστημα χρονικού βήματος μπορεί να χρησιμοποιηθεί μόνο του ή με τις παραμέτρους Συχνότητα επανάληψης χρονικού βήματος ή Χρόνος για ευθυγράμμιση των χρονικών βημάτων.

Αν θέλατε να δημιουργήσετε χρονικά τμήματα που λαμβάνουν χώρα κάθε Δευτέρα από τις 9:00 μέχρι τις 10:00 π.μ., το Διάστημα χρονικού βήματος θα έπρεπε να είναι 1 ώρα, η Συχνότητα επανάληψης χρονικού βήματος θα έπρεπε να είναι 1 εβδομάδα και ο Χρόνος για ευθυγράμμιση των χρονικών βημάτων θα έπρεπε να είναι 9:00:00 π.μ. τη Δευτέρα.

Συχνότητα επανάληψης χρονικού βήματος

Το βήμα που χρησιμοποιείται για τον υπολογισμό ενός χρονικού τμήματος. Η Συχνότητα επανάληψης χρονικού βήματος μπορεί να χρησιμοποιηθεί μόνη της ή με τις παραμέτρους Διάστημα χρονικού βήματος και Χρόνος αναφοράς ή με τις δύο παραμέτρους μαζί Διάστημα χρονικού βήματος και Χρόνος για ευθυγράμμιση των χρονικών βημάτων.

Αν θέλατε να δημιουργήσετε χρονικά τμήματα που λαμβάνουν χώρα κάθε Δευτέρα από τις 9:00 μέχρι τις 10:00 π.μ., το Διάστημα χρονικού βήματος θα έπρεπε να είναι 1 ώρα, η Συχνότητα επανάληψης χρονικού βήματος θα έπρεπε να είναι 1 εβδομάδα και ο Χρόνος για ευθυγράμμιση των χρονικών βημάτων θα έπρεπε να είναι 9:00:00 π.μ. τη Δευτέρα.

Χρόνος για ευθυγράμμιση των χρονικών βημάτων

Η ημερομηνία και η ώρα που χρησιμοποιούνται για ευθυγράμμιση της χρονικής κατάτμησης. Η χρονική κατάτμηση θα εκκινήσει και θα συνεχίσει αντίστροφα από αυτήν την ώρα. Αν δεν έχει επιλεγεί χρόνος αναφοράς, η χρονική κατάτμηση θα ευθυγραμμιστεί στην 1η Ιανουαρίου 1970.

Αν θέλατε να δημιουργήσετε χρονικά τμήματα που λαμβάνουν χώρα κάθε Δευτέρα από τις 9:00 μέχρι τις 10:00 π.μ., το Διάστημα χρονικού βήματος θα έπρεπε να είναι 1 ώρα, η Συχνότητα επανάληψης χρονικού βήματος θα έπρεπε να είναι 1 εβδομάδα και ο Χρόνος για ευθυγράμμιση των χρονικών βημάτων θα έπρεπε να είναι 9:00:00 π.μ. τη Δευτέρα.

Προσθήκη στατιστικών (προαιρετικά)

Μπορείτε να υπολογίσετε στατιστικά για τα στοιχεία που θα συνοψιστούν. Σε αριθμητικά πεδία, μπορείτε να υπολογίσετε τα εξής:

  • Πλήθος—Υπολογίζει τον αριθμό τιμών που δεν είναι null. Μπορεί να χρησιμοποιηθεί σε αριθμητικά πεδία ή σε συμβολοσειρές. Το πλήθος των [null, 0, 2] είναι 2.
  • Άθροισμα—Το άθροισμα των αριθμητικών τιμών σε ένα πεδίο. Το άθροισμα των [null, null, 3] είναι 3.
  • Μέση τιμή—Η μέση τιμή αριθμητικών τιμών. Η μέση τιμή των [0, 2, null] είναι 1.
  • Ελάχιστο—Η ελάχιστη τιμή ενός αριθμητικού πεδίου. Το ελάχιστο των [0, 2, null] είναι 0.
  • Μέγιστο—Η μέγιστη τιμή ενός αριθμητικού πεδίου. Η μέγιστη τιμή των [0, 2, null] είναι 2.
  • Εύρος—Το εύρος ενός αριθμητικού πεδίου. Αυτό υπολογίζεται ως οι ελάχιστες τιμές που αφαιρούνται από την μέγιστη τιμή. Το εύρος των [0, null, 1] είναι 1. Το εύρος των [null, 4] είναι 0.
  • Διακύμανση—Η διακύμανση ενός αριθμητικού πεδίου σε ένα ίχνος. Η διακύμανση του [1] είναι null. Η διακύμανση των [null, 1, 1, 1] είναι 1.
  • Τυπική απόκλιση—Η τυπική απόκλιση ενός αριθμητικού πεδίου. Η τυπική απόκλιση του [1] είναι null. Η τυπική απόκλιση των [null, 1, 1, 1] είναι 1.

Σε πεδία συμβολοσειράς, μπορείτε να υπολογίσετε τα εξής:

  • Πλήθος—Ο αριθμός των συμβολοσειρών που δεν είναι null.
  • Οποιαδήποτε—Αυτό το στατιστικό στοιχείο είναι ένα τυχαίο δείγμα μιας τιμής συμβολοσειράς στο καθορισμένο πεδίο.
Όλα τα στατιστικά στοιχεία υπολογίζονται σε τιμές που δεν είναι null. Το εξαγόμενο θεματικό επίπεδο θα περιέχει ένα νέο πεδίο για κάθε υπολογιζόμενο στατιστικό στοιχείο. Μπορεί να προστεθεί οποιοσδήποτε αριθμός στατιστικών στοιχείων, επιλέγοντας γνώρισμα και στατιστικό στοιχείο.

Χωρική αναφορά (wkid)

Είναι μια προσωρινή παράμετρος για την έκδοση prerelease που ορίζει τη χωρική αναφορά της επεξεργασίας. Πολλά εργαλεία big data απαιτούν τη χρήση ενός προβολικού συστήματος συντεταγμένων ως τη χωρική αναφορά της επεξεργασίας. Από προεπιλογή, το εργαλείο θα χρησιμοποιεί το σύστημα συντεταγμένων εισόδου αλλά θα αποτύχει εάν πρόκειται για σύστημα γεωγραφικών συντεταγμένων. Για να ορίσετε ένα προβολικό σύστημα συντεταγμένων, εισαγάγετε το WKID. Για παράδειγμα, για το Web Mercator πρέπει να εισαγάγετε 3857.

Επιλέξτε ένα ArcGIS Data Store για αποθήκευση των αποτελεσμάτων

Τα αποτελέσματα GeoAnalytics αποθηκεύονται σε ένα ArcGIS Data Store και εκτίθενται ως ένα θεματικό επίπεδο στοιχείων στο Portal for ArcGIS. Στις περισσότερες περιπτώσεις, τα αποτελέσματα πρέπει να αποθηκεύονται στο χωρο-χρονικό data store και αυτό αποτελεί την προεπιλογή. Σε μερικές περιπτώσεις, η αποθήκευση αποτελεσμάτων σε σχεσιακό data store είναι μια καλή επιλογή. Οι ακόλουθοι είναι κάποιοι λόγοι για τους οποίους θα αποθηκεύατε τα αποτελέσματα σε σχεσιακό data store:

  • Χρήση αποτελεσμάτων σε portal για συνεργασία portal.
  • Ενεργοποίηση δυνατοτήτων συγχρονισμού με τα αποτελέσματά σας.

Δεν πρέπει να χρησιμοποιείτε το σχεσιακό data store αν περιμένετε αύξηση των αποτελεσμάτων GeoAnalytics και επιθυμείτε να εκμεταλλευτείτε τις δυνατότητες του χωρο-χρονικού big data store για διαχείριση μεγάλων όγκων δεδομένων.

Όνομα εξαγόμενου θεματικού επιπέδου

Αυτό είναι το όνομα του θεματικού επιπέδου που θα δημιουργηθεί στην ενότητα Το περιεχόμενό μου και θα προστεθεί στο χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο υπάρχει ήδη, το εργαλείο θα αποτύχει.

Με τη χρήση του πτυσσόμενου πλαισίου Αποθήκευση αποτελέσματος σε, μπορείτε να καθορίσετε το όνομα ενός φακέλου στην ενότητα Το περιεχόμενό μουόπου θα αποθηκευτεί το αποτέλεσμα.
