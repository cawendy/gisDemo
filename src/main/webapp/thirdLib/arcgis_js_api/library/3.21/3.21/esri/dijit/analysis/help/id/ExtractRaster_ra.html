Ekstrak Raster

Alat ini dapat menempelkan raster ke perbatasan, baik pada area persegi panjang atau bentuk yang Anda definisikan secara interaktif pada layar. Anda dapat menempelkan cakupan area yang saat ini Anda tampilkan di peta Anda atau dekat area studi yang ditentukan oleh sebuah poligon.

Pilih data input untuk diekstrak

Layer input untuk dipotong.

Tentukan geometri penempelan.

Pilih tipe jangkauan yang akan digunakan ketika memotong geometri di layer:

  • Gunakan jangkauan saat ini—Jangkauan peta saat ini akan digunakan untuk menempelkan layer yang dihasilkan. Ini adalah default.
  • Pilih area studi—Memungkinkan Anda untuk memilih apakah hasil akan menunjukkan area di dalam jangkauan geometri atau area di luar jangkauan geometri. Jika Anda memilih opsi Data di dalam geometri, citra di luar jangkauan akan dihapus. Jika Anda memilih opsi Data di luar geometri, citra di dalam jangkauan akan dihapus. Anda pun dapat mendefinisikan geometri pemotongan dengan menggunakan alat gambar.

Nama layer hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Menggunakan kotak drop-down Simpan hasil di, Anda dapat menentukan nama folder di Konten Saya sebagai tempat penyimpanan hasil.
