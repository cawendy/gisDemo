Konversi Raster ke Fitur

Alat ini mengubah raster ke rangkaian data fitur, seperti titik, garis, atau poligon.

Parameter Pilih kolom memungkinkan Anda untuk memilih kolom atribut manakah dari rangkaian data raster input yang akan menjadi sebuah atribut di kelas fitur output. Parameter ini hanya tersedia untuk raster bilangan bulat yang memiliki atribut tambahan di tabel terkait.

Raster input dapat memiliki ukuran sel berapa pun.

Raster bilangan bulat dapat dikonversi ke fitur titik, garis, atau poligon; tetapi, raster bilangan titik mengambang hanya dapat dikonversi ke fitur titik.

Jika Gunakan jangkauan peta saat ini dicentang, hanya sel raster yang terlihat di dalam jangkauan peta saat ini yang akan dikonversikan. Jika tidak dicentang, semua sel pada layer raster akan dikonversikan, meskipun berada di luar jangkauan peta saat ini. Perhatikan hal ini jika layer raster memiliki ukuran yang besar atau memiliki resolusi yang sangat tinggi.

Contoh penggunaan alat ini mencakup hal berikut:

  • Membuat fitur poligon dari raster klasifikasi penggunaan tanah.
  • Aliran yang teridentifikasi di DEM dapat dikonversikan ke fitur garis.
  • DEM dapat dikonversikan ke fitur titik yang dapat digunakan di alur kerja berbasis nonraster.

Pilih layer raster untuk dikonversi

Layer raster input yang akan dikonversikan ke feature layer.

Pilih kolom

Sediakan kolom yang menentukan nilai manakah yang akan digunakan untuk konversi.

Bisa saja berupa bilangan bulat atau nilai teks.

Kolom yang berisi nilai bilangan titik mengambang hanya dapat digunakan jika output adalah ke rangkaian data titik.

Pilihan default adalah menggunakan kolom Nilai, yang berisi nilai di setiap sel raster.

Pilih tipe output

Pilih tipe output antara titik, garis, atau poligon.

  • Titik—Raster akan dikonversi ke rangkaian data titik.
  • Garis—Raster akan dikonversi ke rangkaian data fitur garis.
  • Poligon—Raster akan dikonversi ke rangkaian data fitur poligon.

Sederhanakan garis atau poligon

Tentukan apakah fitur harus disederhanakan.

Jika dicentang (default), fitur garis atau poligon akan diperhalus untuk memberikan hasil yang lebih digeneralisasi. Jika opsi ini tidak dicentang, fitur linear akan mengikuti batasan sel rangkaian data raster.

Nama layer hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Menggunakan kotak drop-down Simpan hasil di, Anda dapat menentukan nama folder di Konten Saya sebagai tempat penyimpanan hasil.
