Σύνοψη γνωρισμάτων

Το εργαλείο συνοψίζει όλες τις τιμές που συμφωνούν σε ένα ή περισσότερα πεδία και υπολογίζει στατιστικά για αυτές. Το βασικότερο στατιστικό είναι το πλήθος των στοιχείων που έχουν συνοψιστεί μαζί, αλλά μπορείτε να υπολογίσετε και άλλα πιο σύνθετα στατιστικά στοιχεία.

Για παράδειγμα, ας υποθέσουμε ότι έχετε σημειακά στοιχεία τοποθεσιών καταστημάτων με ένα πεδίο που αντιστοιχεί στο DISTRICT_MANAGER_NAME και θέλετε να συνοψίσετε τις πωλήσεις ανά διευθυντή. Μπορείτε να προσδιορίσετε το πεδίο DISTRICT_MANAGER_NAME ως το πεδίο για ομαδοποίηση, και θα συνοψιστούν όλες οι σειρές δεδομένων που αντιστοιχούν σε μεμονωμένους διευθυντές. Αυτό σημαίνει ότι όλες οι τοποθεσίες καταστημάτων που διαχειρίζεται ο Διευθυντής1 θα συνοψιστούν σε μία σειρά με υπολογισμένα συνοπτικά στατιστικά στοιχεία. Σε αυτό το παράδειγμα, θα υπολογίζονταν στατιστικά όπως το πλήθος των καταστημάτων και το άθροισμα των TOTAL_SALES για όλα τα καταστήματα που διαχειρίζεται ο Διευθυντής1 όπως και κάθε άλλος διευθυντής στο πεδίο DISTRICT_MANAGER_NAME.

Διαλέξτε τα δεδομένα εισαγωγής προς σύνοψη

Το θεματικό επίπεδο που περιέχει τα πεδία που θα συνοψιστούν.

Εκτός από την επιλογή ενός θεματικού επιπέδου από το χάρτη σας, μπορείτε επίσης να επιλέξετε Αναζήτηση θεματικών επιπέδων στο κάτω μέρος της αναπτυσσόμενης λίστας για αναζήτηση στα περιεχόμενά σας ενός κοινόχρηστου αρχείου συνόλου δεδομένων με big data ή ενός feature layer.

Διαλέξτε τα πεδία προς σύνοψη

Λίστα πεδίων από το θεματικό επίπεδο εισόδου. Επιλεγμένα πεδία θα συνοψιστούν και θα χρησιμοποιηθούν για υπολογισμό στατιστικών στοιχείων. Για παράδειγμα, αν διαλέγατε ένα πεδίο που είχε τίτλο PropertyType με τις τιμές των εμπορικών και οικιστικών πεδίων, θα συνοψίζονταν όλα τα οικιστικά πεδία μαζί, με υπολογισμένα συνοπτικά στατιστικά στοιχεία, και όλα τα εμπορικά πεδία μαζί.

Προσθήκη στατιστικών (προαιρετικά)

Μπορείτε να υπολογίσετε στατιστικά για τα στοιχεία που θα συνοψιστούν. Σε αριθμητικά πεδία, μπορείτε να υπολογίσετε τα εξής:

  • Πλήθος—Υπολογίζει τον αριθμό τιμών που δεν είναι null. Μπορεί να χρησιμοποιηθεί σε αριθμητικά πεδία ή σε συμβολοσειρές. Το πλήθος των [null, 0, 2] είναι 2.
  • Άθροισμα—Το άθροισμα των αριθμητικών τιμών σε ένα πεδίο. Το άθροισμα των [null, null, 3] είναι 3.
  • Μέση τιμή—Η μέση τιμή αριθμητικών τιμών. Η μέση τιμή των [0, 2, null] είναι 1.
  • Ελάχιστο—Η ελάχιστη τιμή ενός αριθμητικού πεδίου. Το ελάχιστο των [0, 2, null] είναι 0.
  • Μέγιστο—Η μέγιστη τιμή ενός αριθμητικού πεδίου. Η μέγιστη τιμή των [0, 2, null] είναι 2.
  • Εύρος—Το εύρος ενός αριθμητικού πεδίου. Αυτό υπολογίζεται ως οι ελάχιστες τιμές που αφαιρούνται από την μέγιστη τιμή. Το εύρος των [0, null, 1] είναι 1. Το εύρος των [null, 4] είναι 0.
  • Διακύμανση—Η διακύμανση ενός αριθμητικού πεδίου σε ένα ίχνος. Η διακύμανση του [1] είναι null. Η διακύμανση των [null, 1, 1, 1] είναι 1.
  • Τυπική απόκλιση—Η τυπική απόκλιση ενός αριθμητικού πεδίου. Η τυπική απόκλιση του [1] είναι null. Η τυπική απόκλιση των [null, 1, 1, 1] είναι 1.

Σε πεδία συμβολοσειράς, μπορείτε να υπολογίσετε τα εξής:

  • Πλήθος—Ο αριθμός των συμβολοσειρών που δεν είναι null.
  • Οποιαδήποτε—Αυτό το στατιστικό στοιχείο είναι ένα τυχαίο δείγμα μιας τιμής συμβολοσειράς στο καθορισμένο πεδίο.
Όλα τα στατιστικά στοιχεία υπολογίζονται σε τιμές που δεν είναι null. Το εξαγόμενο θεματικό επίπεδο θα περιέχει ένα νέο πεδίο για κάθε υπολογιζόμενο στατιστικό στοιχείο. Μπορεί να προστεθεί οποιοσδήποτε αριθμός στατιστικών στοιχείων, επιλέγοντας γνώρισμα και στατιστικό στοιχείο.

Επιλογή χώρου αποθήκευσης δεδομένων

Τα αποτελέσματα GeoAnalytics αποθηκεύονται σε ένα ArcGIS Data Store και εκτίθενται ως ένα θεματικό επίπεδο στοιχείων στο Portal for ArcGIS. Στις περισσότερες περιπτώσεις, τα αποτελέσματα πρέπει να αποθηκεύονται στο χωρο-χρονικό data store και αυτό αποτελεί την προεπιλογή. Σε μερικές περιπτώσεις, η αποθήκευση αποτελεσμάτων σε σχεσιακό data store είναι μια καλή επιλογή. Οι ακόλουθοι είναι κάποιοι λόγοι για τους οποίους θα αποθηκεύατε τα αποτελέσματα σε σχεσιακό data store:

  • Χρήση αποτελεσμάτων σε portal για συνεργασία portal.
  • Ενεργοποίηση δυνατοτήτων συγχρονισμού με τα αποτελέσματά σας.

Όνομα εξαγόμενου θεματικού επιπέδου

Αυτό είναι το όνομα του θεματικού επιπέδου που θα δημιουργηθεί στην ενότητα Το περιεχόμενό μου και θα προστεθεί στο χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο υπάρχει ήδη, το εργαλείο θα αποτύχει.

Με τη χρήση του πτυσσόμενου πλαισίου Αποθήκευση αποτελέσματος σε, μπορείτε να καθορίσετε το όνομα ενός φακέλου στην ενότητα Το περιεχόμενό μουόπου θα αποθηκευτεί το αποτέλεσμα.
