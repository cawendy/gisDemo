تحويل معلم إلى بيانات نقطية

يعمل هذا القناع على تحويل المعالم إلى مجموعة بيانات نقطية.

يمكن تحويل أي تصنيف معلم يحتوي على معالم نقطية أو خطية أو مضلعة إلى مجموعة بيانات نقطية.

يستخدم مركز الخلية لتحديد قيمة بيكسل البيانات النقطية الناتجة.

يُحدد نوع حقل المدخلات نوع البيانات النقطية الناتجة. إذا كان الحقل عبارة عن عدد صحيح، ستكون البيانات النقطية الناتجة عدد صحيح؛ وإذا كان عبارة عن نقطة عائمة، ستكون المخرجات عبارة عن نقطة عائمة.

اختر طبقة معالم لتحويلها

طبقة المعلم المدخلة المراد تحويلها إلى مجموعة بيانات نقطية.

اختر حقلاً

اختر الحقل الذي يُستخدم لتعيين القيم إلى البيانات النقطية الناتجة.

حجم خلية مُخرج

إدخال حجم الخلية ووحدة البيانات النقطية الناتجة.

الوحدات المتوفرة هي قدم، أميال، أمتار، وكيلومتر.

اسم طبقة النتيجة

سيتم إنشاء اسم الطبقة في المحتوى وسيتم إضافته إلى الخريطة. يستند الاسم الافتراضي إلى اسم الأداة واسم الطبقة المدخلة. إذا وُجدت الطبقة بالفعل، سيطلب منك كتابة اسم آخر.

عند استخدام مربع السهم المنسدل حفظ النتائج في يمكن تحديد اسم المجلد في المحتويات حيث يتم حفظ النتائج.
