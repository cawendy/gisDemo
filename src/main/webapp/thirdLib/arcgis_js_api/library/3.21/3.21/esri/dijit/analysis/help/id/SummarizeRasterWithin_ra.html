Ringkas Raster di Dalam

Alat ini meringkas raster berdasarkan area (zona) yang ditentukan oleh layer input pertama.

Beberapa aplikasi contoh termasuk hal berikut:

  • Dengan mengetahui batas air dan layer batas penggunaan lahan menurut jenis penggunaan lahan, hitung luas total dari jenis penggunaan lahan untuk setiap batas air.
  • Dengan mengetahui layer bagian persil di suatu daerah dan layer batas kota, ringkaslah nilai rata-rata dari bagian persil yang kosong di dalam setiap batas kota.

Jika Gunakan jangkauan peta saat ini dicentang, hanya fitur-fitur pada layer input dan layer yang akan diringkas yang terlihat di jangkauan peta saat ini yang akan dianalisis. Jika tidak dicentang, semua fitur pada layer input dan layer yang akan diringkas akan dianalisis meskipun berada di luar jangkauan peta saat ini.

Pilih area layer untuk meringkas layer raster di dalam batas yang ditentukan

Layer yang menentukan perbatasan area atau zona yang akan diringkas.

Layer dapat berupa raster atau data fitur. Untuk raster, zona ditentukan oleh semua lokasi di input yang memiliki nilai sel yang sama. Area itu tidak harus menular.

Pilih kolom untuk menentukan batas

Ini adalah artibut layer yang akan digunakan untuk menentukan perbatasan area.

Contoh, misalnya layer input pertama menentukan batas unit manajemen, dengan atribut yang menentukan wilayah, distirk, dan ID parsel setiap unit. Anda pun memiliki layer raster yang menentukan indeks hayati untuk setiap lokasi. Dengan kolom yang Anda pilih, Anda dapat memutuskan untuk menghitung rata-rata hayati di tingkat lokal, distrik, atau wilayah.

Pilih layer raster untuk diringkas

Sel raster di layer ini akan diringkas berdasarkan area (zona) tempat posisi sel tersebut.

Pilih statistik yang akan dihitung

Setelah Anda memilih layer untuk diringkas, Anda dapat memilih untuk menghitung statistik dari setiap atribut numerik dari titik, garis, atau area di dalam layer area input.

Tipe statistik yang tersedia ketika kolom yang dipilih adalah bilangan bulat adalah Rata-rata, Maksimal, Nilai Media, Minimal, Minoritas, Rentang, Standar deviasi, Jumlah, dan Keragaman.

Jika kolom adalah titik melayang, opsinya adalah Rata-rata, Maksimal, Minimal, Rentang, Standar deviasi, dan Jumlah.

Abaikan nilai yang hilang dalam penghitungan

Jika Anda memilih untuk mengabaikan nilai yang hilang, hanya sel yang memiliki nilai di dalam layer yang akan diringkas yang akan digunakan untuk menentukan nilai output untuk area tersebut. Jika tidak, jika ada nilai yang hilang di mana pun di dalam area, maka akan diasumsikan bahwa informasi tidak memadai untuk melakukan perhitungan statistik untuk semua sel di zona itu, dan area itu akan menerima nilai null (NoData) di output.

Nama layer hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Menggunakan kotak drop-down Simpan hasil di, Anda dapat menentukan nama folder di Konten Saya sebagai tempat penyimpanan hasil.
