Convertire obiect spaţial în raster

Această sarcină converteşte obiectele spaţiale într-un set de date raster.

Orice obiect spaţial care conţine puncte, linii sau poligoane poate fi convertit în set de date raster.

Centrul celulei este utilizat pentru a decide valoarea pixelului rasterului de ieşire.

Tipul câmpului de intrare determină tipul de raster de ieşire. Dacă câmpul este un număr întreg, rasterul de ieşire va fi un număr întreg; dacă este un punct mobil, ieşirea va fi un punct mobil.

Alegeţi stratul tematic de obiecte spaţiale pentru convertire

Stratul tematic de obiecte spaţiale de intrare care va fi convertit într-un set de date raster.

Alegere câmp

Alegeţi câmpul care va fi utilizat pentru a atribui valori rasterului de ieşire.

Dimensiune celulă ieşire

Introduceţi dimensiunea celulei şi unitatea pentru rasterele de ieşire.

Unităţile disponibile sunt Ft, Mile, Metri şi Kilometri.

Nume strat tematic rezultat

Acesta este numele stratului tematic ce va fi creat în Resursele mele şi va fi adăugat pe hartă. Numele implicit se bazează pe numele instrumentului şi numele stratului tematic de intrare. Dacă stratul tematic rezultat există deja, vi se va solicita să furnizaţi un alt nume.

Utilizând caseta derulantă Salvare rezultat în, puteţi specifica numele unui director din Resursele mele în care va fi salvat rezultatul.
