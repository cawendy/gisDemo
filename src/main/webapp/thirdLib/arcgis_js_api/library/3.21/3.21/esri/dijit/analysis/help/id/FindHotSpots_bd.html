Temukan Hot Spot

Alat Temukan Hot Spot akan menentukan apakah ada kluster yang signifikan secara statistik pada pola spasial data Anda.

  • Apakah titik (kejadian kriminal, pohon, atau kecelakaan lalu lintas) Anda benar-benar terkluster? Bagaimana Anda bisa yakin?
  • Apakah Anda benar-benar menemukan hot spot yang signifikan secara statistik, atau apakah peta Anda memberikan cerita yang berbeda jika Anda mengubah cara simbolisasinya?
Alat Temukan Hot Spot akan membantu Anda menjawab pertanyaan tersebut dengan yakin.

Bahkan pola spasial acak memiliki beberapa derajat kluster. Sebagai tambahan, mata dan otak kita secara alami mencoba menemukan pola bahkan ketika pola tersebut sebenarnya tidak ada. Maka dari itu, akan menjadi sulit untuk mengetahui apakah pola pada data Anda merupakan hasil kerja nyata dari proses spasial, atau hanya merupakan hasil kemungkinan acak. Hal tersebut menjadi alasan mengapa para peneliti dan analis menggunakan metode statistik seperti Temukan Hot Spot (Getis-Ord Gi*) untuk melakukan kuantifikasi pola spasial. Ketika Anda menemukan pengklusteran secara statistik yang signifikan pada data Anda, Anda memiliki informasi yang berharga. Dengan mengetahui tempat dan waktu pengklusteran terjadi, akan dapat memberikan petunjuk penting tentang proses yang mengakibatkan pola yang Anda lihat. Dengan mengetahui bahwa perampokan rumah, sebagai contoh, secara konsisten lebih tinggi pada lingkungan tertentu, merupakan informasi penting jika Anda akan mendesain strategi pencegahan yang efektif, mengalokasikan sumber daya polisi yang sedikit, memulai program sistem keamanan lingkungan, mengotorisasi investigasi kriminal mendalam, atau untuk mengidentifikasi tersangka kriminal.

Pilih layer tempat hot spot akan dihitung

Layer titik tempat menemukan spot panas atau dingin.

Analisis ini menggunakan bin dan memerlukan sistem koordinat terproyeksi. Anda dapat menetapkan Sistem koordinat pemrosesan dalam Lingkungan Analisis. Jika sistem koordinat pemrosesan Anda tidak ditetapkan ke sistem koordinat terproyeksi, Anda akan diminta untuk menetapkannya saat Anda Jalankan Analisis.

Selain memilih layer dari peta Anda, Anda dapat memilih Telusuri Layer yang terdapat di bagian paling bawah daftar drop-down ke konten Anda untuk set data yang dibagikan atau feature layer dari big data.

Temukan kluster tinggi dan rendah

Analisis ini menjawab pertanyaan: Dimanakah nilai tinggi dan rendah terkluster secara spasial?

Jika data Anda berupa titik dan Anda memilih Jumlah Titik, alat ini akan mengevaluasi pengaturan spasial fitur-fitur titik untuk menjawab pertanyaan: Di manakah titik-titik tersebut terkluster atau tersebar tidak secara semestinya?

Jika Anda memilih kolom, alat ini akan mengevaluasi pengaturan spasial dari nilai-nilai yang terkait dengan masing-masing fitur untuk menjawab pertanyaan: Dimanakah pengklasteran nilai tinggi dan rendah?

Pilih ukuran bin untuk agregasi

Jarak yang digunakan untuk membuat square bin yang akan digunakan untuk menganalisis titik input Anda.

Cari hot spot menggunakan langkah waktu (opsional)

Jika waktu diaktifkan pada layer titik dan merupakan instan tipe, Anda dapat menganalisis menggunakan langkah waktu.

Intervalo langkah waktu

Interval waktu yang digunakan untuk menghasilkan langkah waktu. Waktu dapat diselaraskan pada waktu mulai atau waktu akhir dari data input atau pada waktu rujukan tertentu.

Interval langkah waktu

Interval waktu yang digunakan untuk menghasilkan langkah waktu. Waktu dapat diselaraskan pada waktu mulai atau waktu akhir dari data input atau pada waktu rujukan tertentu.

Pilih cara menyelaraskan langkah waktu

Bagaimana langkah waktu diselaraskan. Ada tiga cara untuk menyelaraskan langkah waktu:

  • Waktu Mulai—Langkah waktu diselaraskan dengan fitur pertama secara tepat.
  • Waktu Selesai—Langkah waktu diselaraskan dengan fitur terakhir secara tepat.
  • Waktu Referensi—Langkah waktu diselaraskan dengan waktu yang ditetapkan.

Waktu referensi untuk diselaraskan dengan langkah waktu

Tanggal dan waktu yang digunakan untuk menyelaraskan langkah waktu.

Pilih ukuran lingkungan untuk perhitungan hot spot

Jarak yang digunakan untuk menentukan lingkungan akan digunakan untuk perhitungan hot spot. Lingkungan seharusnya lebih besar dari ukuran bin untuk memastikan bahwa setiap bin memiliki setidaknya satu tetangga. Setiap bin akan dianalisis dan dibandingkan dengan bin yang berdampingan.

SpatialReference (wkid)

Ini adalah parameter sementara untuk pra-rilis untuk menetapkan referensi spasial pemrosesan. Banyak alat big data yang mewajibkan bahwa sistem koordinat proyeksi digunakan sebagai referensi spasial untuk pemrosesan. Secara default, alat akan menggunakan sistem koordinat input, tetapi akan gagal jika ini adalah sistem koordinat geografik. Untuk menetapkan sistem koordinat proyeksi, masukkan WKID. Misalnya, Web Mercator harus dimasukkan sebagai 3857.

Pilih ArcGIS Data Store untuk penyimpanan hasil

Hasil GeoAnalytics disimpan di ArcGIS Data Store dan ditampilkan sebagai layer fitur di Portal for ArcGIS. Di banyak kasus, hasil harus disimpan di data store spatiotemporal dan ini adalah pilihan default. Di banyak kasus, menyimpan hasil data store relasional adalah opsi yang bagus. Berikut adalah alasan mengapa Anda mungkin ingin menyimpan hasil di data store relasional:

  • Gunakan hasil di kolaborasi portal-ke-portal
  • Aktifkan kemampuan sinkronisasi dengan hasil Anda.

Anda tidak boleh menggunakan data store relasional jika Anda berharap hasil GeoAnalytics untuk ditingkatkan, dan ingin memanfaatkan kemampuan big data store spatiotemporal untuk menangani jumlah data yang besar.

Nama layer hasil

Ini adalah nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer sudah ada, alat tidak akan berfungsi.

Layer hasil ini akan memperlihatkan kepada Anda kluster yang signifikan secara statistik dengan nilai tinggi atau rendah atau jumlah titik. Jika nama layer hasil sudah ada, Anda akan diminta untuk memberi nama baru.

Menggunakan kotak drop-down Simpan hasil di, Anda dapat menentukan nama folder di Konten Saya sebagai tempat penyimpanan hasil.
