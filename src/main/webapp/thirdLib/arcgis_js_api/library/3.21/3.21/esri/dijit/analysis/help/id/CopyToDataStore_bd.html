Salin ke Data Store

Alat ini akan mengambil layer input dan menyalinnya ke data store. Data akan disalin ke ArcGIS Data Store dan disimpan di tempat data store relasional atau spatiotemporal Anda.

Contoh, Anda dapat menyalin fitur yang disimpan di berbagi file big data ke data store relasional dan menentukan bahwa hanya fitur di dalam jangkauan peta saat ini yang akan disalin. Hal ini akan membuat hosted feature service hanya dengan fitur yang berada di dalam jangkauan peta yang telah ditentukan.

Pilih layer untuk disalin

Layer atau tabel yang akan disalin.

Selain memilih layer dari peta Anda, Anda dapat memilih Telusuri Layer yang terdapat di bagian paling bawah daftar drop-down ke konten Anda untuk set data yang dibagikan atau feature layer dari big data.

Anda dapat menggunakan semua fitur atau membuat pilihan (opsional)

Atau, gunakan tombol pemilih untuk mengidentifikasi lokasi yang akan disalin.

Pilih ArcGIS Data Store untuk disalin ke

Data store tempat layer akan disalin. Parameter ini akan mengabaikan pilihan dalam Lingkungan Analisis Opsinya adalah sebagai berikut:

  • Spatiotemporal—Ini adalah defaultnya.
  • Relasional.

Nama layer hasil

Ini adalah nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer sudah ada, alat tidak akan berfungsi.

Menggunakan kotak drop-down Simpan hasil di, Anda dapat menentukan nama folder di Konten Saya sebagai tempat penyimpanan hasil.
