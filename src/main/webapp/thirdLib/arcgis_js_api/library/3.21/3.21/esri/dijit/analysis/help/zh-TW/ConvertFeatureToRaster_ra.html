將圖徵轉換為光柵

此任務可將圖徵轉換成點陣資料集。

可將包含點、線或多邊形圖徵的任何圖徵類別轉換成點陣資料集。

儲存格中心可用來決定輸出點陣像素的值。

輸入欄位類型可決定輸出點陣的類型。如果欄位為整數，輸出點陣將是整數；如果為浮點，輸出將是浮點。

選擇要轉換的圖徵圖層

要轉換為點陣資料集的輸入圖徵圖層。

選擇欄位

選擇會將值指派給輸出點陣的欄位。

輸出儲存格大小

針對輸出點陣輸入儲存格大小和單位。

可用的單位為英呎、英哩、公尺和公里。

結果圖層名稱

在 我的內容中建立並新增到地圖中的圖層的名稱。預設名稱參照工具名稱以及輸入圖層名稱。如果該名稱的圖層已存在，則系統將要求您提供其他名稱。

使用 將結果儲存到下拉式方塊，可以在 我的內容中指定储存结果的文件夾的名稱。
