Εύρεση Hot Spot

Το εργαλείο "Εύρεση Hot Spot" προσδιορίζει αν υπάρχει κάποια στατιστικά σημαντική συστάδα στο χωρικό μοτίβο των δεδομένων σας.

  • Είναι πράγματι τα σημεία σας (εγκληματικές ενέργειες, δέντρα ή τροχαία ατυχήματα) συγκεντρωμένα σε συστάδες; Πώς μπορείτε να είστε βέβαιοι;
  • Ανακαλύψατε ένα στατιστικά σημαντικό Hot Spot ή μήπως ο χάρτης θα αποκάλυπτε κάτι άλλο, αν αλλάζατε τον τρόπο απεικόνισής του;
Με το εργαλείο "Εύρεση Hot Spot" θα μπορέσετε να απαντήσετε σε αυτές τις ερωτήσεις με βεβαιότητα.

Ακόμη και τα τυχαία χωρικά μοτίβα παρουσιάζουν κάποιο βαθμό συστάδας. Επιπλέον, τα μάτια μας και το μυαλό μας προσπαθούν φυσιολογικά να βρουν μοτίβα ακόμη και όταν δεν υπάρχουν. Συνεπώς, είναι δύσκολο να γνωρίζετε αν τα μοτίβα στα δεδομένα σας είναι το αποτέλεσμα πραγματικών χωρικών διεργασιών ή απλώς ένα τυχαίο αποτέλεσμα. Για το λόγο αυτό, οι ερευνητές και οι αναλυτές χρησιμοποιούν στατιστικές μεθόδους όπως η Εύρεση Hot Spot (Getis-Ord Gi*) για την ποσοτικοποίηση χωρικών μοτίβων. Όταν βρίσκετε σημαντικές συστάδες, από την άποψη της στατιστικής, στα δεδομένα σας, έχετε πολύτιμες πληροφορίες. Η γνώση του πού και πώς παρουσιάζονται συστάδες, μπορεί να παρέχει σημαντικά στοιχεία σχετικά με τις διεργασίες που προωθούν τα μοτίβα που βλέπετε. Η γνώση ότι οι διαρρήξεις σε σπίτια, για παράδειγμα, είναι σταθερά περισσότερες σε συγκεκριμένες συνοικίες, αποτελεί πολύ σημαντική πληροφορία όταν θέλετε να σχεδιάσετε αποτελεσματικές στρατηγικές αποτροπής, να κατανείμετε τους λιγοστούς πόρους αστυνόμευσης, να ξεκινήσετε προγράμματα παρακολούθησης συνοικιών, να εγκρίνετε λεπτομερείς έρευνες για εγκλήματα ή να προσδιορίσετε πιθανούς υπόπτους.

Επιλέξτε το θεματικό επίπεδο για το οποίο θα υπολογιστούν hot spot

Το σημειακό θεματικό επίπεδο για το οποίο θα βρεθούν hot και cold spot.

Για αυτήν την ανάλυση χρησιμοποιούνται bin και απαιτείται προβολικό σύστημα συντεταγμένων. Μπορείτε να ρυθμίσετε το Σύστημα συντεταγμένων επεξεργασίας στα Περιβάλλοντα ανάλυσης. Εάν το σύστημα συντεταγμένων επεξεργασίας δεν έχει οριστεί σε προβολικό σύστημα συντεταγμένων, θα σας ζητηθεί να το ορίσετε κατά την Εκτέλεση ανάλυσης.

Εκτός από την επιλογή ενός θεματικού επιπέδου από το χάρτη σας, μπορείτε επίσης να επιλέξετε Αναζήτηση θεματικών επιπέδων στο κάτω μέρος της αναπτυσσόμενης λίστας για αναζήτηση στα περιεχόμενά σας ενός κοινόχρηστου αρχείου συνόλου δεδομένων με big data ή ενός feature layer.

Εύρεση συστάδων υψηλής και χαμηλής συγκέντρωσης

Η ανάλυση αυτή απαντά στην ερώτηση: Πού συγκεντρώνονται χωρικά υψηλές και χαμηλές τιμές;

Αν τα δεδομένα σας είναι σημεία και επιλέξετε το Μετρήσεις σημείων, το εργαλείο αυτό θα υπολογίσει τη χωρική κατανομή των σημειακών στοιχείων για να απαντήσει στην ερώτηση: Πού συγκεντρώνονται ή διασκορπίζονται τα σημεία με μη αναμενόμενο τρόπο;

Αν επιλέξετε ένα πεδίο, το εργαλείο αυτό θα υπολογίσει τη χωρική κατανομή των τιμών που σχετίζονται με κάθε στοιχείο για να απαντήσει στην ερώτηση: Πού συγκεντρώνονται υψηλές και χαμηλές τιμές;

Επιλέξτε το μέγεθος bin για τη συνάθροιση

Η απόσταση που χρησιμοποιείται για τη δημιουργία των τετράγωνων bin που θα χρησιμοποιηθούν για την ανάλυση των σημείων εισόδου.

Βρείτε hot spot χρησιμοποιώντας χρονικά βήματα (προαιρετικό)

Αν είναι ενεργοποιημένος ο χρόνος στο σημειακό θεματικό επίπεδο, και ως τύπος έχει επιλεχθεί στιγμή, η ανάλυση μπορεί να γίνει χρησιμοποιώντας χρονικά βήματα.

Διάστημα χρονικού βήματος

Χρονικό διάστημα για τη δημιουργία των χρονικών βημάτων. Μπορεί να ευθυγραμμιστεί με το χρόνο έναρξης ή λήξης των δεδομένων εισόδου ή με έναν καθορισμένο χρόνο αναφοράς.

Διάστημα χρονικού βήματος

Χρονικό διάστημα για τη δημιουργία των χρονικών βημάτων. Μπορεί να ευθυγραμμιστεί με το χρόνο έναρξης ή λήξης των δεδομένων εισόδου ή με έναν καθορισμένο χρόνο αναφοράς.

Επιλέξτε τον τρόπο ευθυγράμμισης των χρονικών βημάτων

Πώς ευθυγραμμίζονται τα χρονικά βήματα. Τα χρονικά βήματα ευθυγραμμίζονται με τρεις τρόπους:

  • Χρόνος έναρξης—Τα χρονικά βήματα ευθυγραμμίζονται με το πρώτο στοιχείο στον χρόνο.
  • Χρόνος λήξης—Τα χρονικά βήματα ευθυγραμμίζονται με το τελευταίο στοιχείο στον χρόνο.
  • Χρόνος αναφοράς—Τα χρονικά βήματα ευθυγραμμίζονται με έναν συγκεκριμένο χρόνο.

Επιλέξτε το μέγεθος γειτονιάς για τους hot spot υπολογισμούς

Η απόσταση που χρησιμοποιείται για τον καθορισμό της γειτονιάς στους υπολογισμούς για hot spot. Η γειτονιά πρέπει να είναι μεγαλύτερη από το μέγεθος του bin ώστε να εξασφαλίζεται ότι κάθε bin έχει τουλάχιστον ένα γειτονικό. Κάθε bin αναλύεται και συγκρίνεται με τα γειτονικά bin.

Χωρική αναφορά (wkid)

Είναι μια προσωρινή παράμετρος για την έκδοση prerelease που ορίζει τη χωρική αναφορά της επεξεργασίας. Πολλά εργαλεία big data απαιτούν τη χρήση ενός προβολικού συστήματος συντεταγμένων ως τη χωρική αναφορά της επεξεργασίας. Από προεπιλογή, το εργαλείο θα χρησιμοποιεί το σύστημα συντεταγμένων εισόδου αλλά θα αποτύχει εάν πρόκειται για σύστημα γεωγραφικών συντεταγμένων. Για να ορίσετε ένα προβολικό σύστημα συντεταγμένων, εισαγάγετε το WKID. Για παράδειγμα, για το Web Mercator πρέπει να εισαγάγετε 3857.

Επιλέξτε ένα ArcGIS Data Store για αποθήκευση των αποτελεσμάτων

Τα αποτελέσματα GeoAnalytics αποθηκεύονται σε ένα ArcGIS Data Store και εκτίθενται ως ένα θεματικό επίπεδο στοιχείων στο Portal for ArcGIS. Στις περισσότερες περιπτώσεις, τα αποτελέσματα πρέπει να αποθηκεύονται στο χωρο-χρονικό data store και αυτό αποτελεί την προεπιλογή. Σε μερικές περιπτώσεις, η αποθήκευση αποτελεσμάτων σε σχεσιακό data store είναι μια καλή επιλογή. Οι ακόλουθοι είναι κάποιοι λόγοι για τους οποίους θα αποθηκεύατε τα αποτελέσματα σε σχεσιακό data store:

  • Χρήση αποτελεσμάτων σε portal για συνεργασία portal.
  • Ενεργοποίηση δυνατοτήτων συγχρονισμού με τα αποτελέσματά σας.

Δεν πρέπει να χρησιμοποιείτε το σχεσιακό data store αν περιμένετε αύξηση των αποτελεσμάτων GeoAnalytics και επιθυμείτε να εκμεταλλευτείτε τις δυνατότητες του χωρο-χρονικού big data store για διαχείριση μεγάλων όγκων δεδομένων.

Όνομα εξαγόμενου θεματικού επιπέδου

Αυτό είναι το όνομα του θεματικού επιπέδου που θα δημιουργηθεί στην ενότητα Το περιεχόμενό μου και θα προστεθεί στο χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο υπάρχει ήδη, το εργαλείο θα αποτύχει.

Αυτό το επίπεδο αποτελεσμάτων θα εμφανίσει στατιστικά σημαντικές συστάδες υψηλών και χαμηλών τιμών ή πλήθους σημείων. Αν το όνομα του εξαγόμενου θεματικού επιπέδου υπάρχει ήδη, θα σας ζητηθεί να το μετονομάσετε.

Με τη χρήση του πτυσσόμενου πλαισίου Αποθήκευση αποτελέσματος σε, μπορείτε να καθορίσετε το όνομα ενός φακέλου στην ενότητα Το περιεχόμενό μουόπου θα αποθηκευτεί το αποτέλεσμα.
