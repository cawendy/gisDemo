Sumarizare atribute

Acest instrument rezumă toate valorile corespunzătoare din unul sau mai multe câmpuri şi calculează statistici pentru acestea. Cea mai simplă statistică este calculul obiectelor spaţiale care au fost rezumate împreună, însă puteţi calcula şi statistici mai avansate.

De exemplu, să presupunem că aveţi obiecte spaţiale punct reprezentând locaţii de magazine cu un câmp DISTRICT_MANAGER_NAME şi doriţi un rezumat al vânzărilor de cafea după manager. Puteţi specifica câmpul DISTRICT_MANAGER_NAME drept câmp de dizolvare şi toate rândurile de date care reprezintă manageri individuali vor fi rezumate. Aceasta înseamnă că toate locaţiile de magazine care sunt gestionate de Manager1 vor fi rezumate într-un rând cu statistici de rezumat calculate. În această situaţie, ar fi calculate statisticile precum numărul de magazine şi suma TOTAL_SALES atât pentru toate magazinele gestionate de Manager1, cât şi pentru orice alt manager menţionat în câmpul DISTRICT_MANAGER_NAME.

Alegeţi intrarea pentru rezumat

Stratul care conţine câmpurile care vor fi rezumate.

În afară de a alege un strat tematic din harta dvs., puteţi selecta opţiunile Răsfoire straturi tematice în partea inferioară a listei verticale pentru a răsfoi conţinutul pentru stratul tematic sau setul de date pentru partajarea de fişiere big data.

Alegeţi câmpurile pentru rezumat

Listă de câmpuri din stratul tematic de intrare. Câmpurile selectate vor fi rezumate şi utilizate pentru a calcula statistici. De exemplu, dacă alegeţi un câmp denumit PropertyType cu valorile pentru comercial şi rezidenţial, toate câmpurile rezidenţiale vor fi rezumate împreună, cu statisticile rezumatului calculate şi toate câmpurile comerciale ar fi rezumate împreună.

Adăugare statistici (opţional)

Puteţi calcula statistici despre obiectele spaţiale rezumate. În câmpurile numerice puteţi calcula următoarele:

  • Numărare—Calculează numărul de valori care nu sunt nule. Poate fi utilizat în câmpurile sau în şirurile numerice. Calculul [nul, 0, 2] este egal cu 2.
  • Sumă - Suma valorilor numerice dintr-un câmp. Suma [nul, 0, 3] este egală cu 3.
  • Media—Media valorilor numerice. Media dintre [0, 2, nul] este 1.
  • Min—Valoarea minimă a unui câmp numeric. Minimul dintre [0, 2, nul] este 0.
  • Max—Valoarea maximă a unui câmp numeric. Valoarea maximă a [0, 2, nul] este 2.
  • Interval—Intervalul unui câmp numeric. Acesta este calculat ca valoare minimă scăzută din valoare maximă. Intervalul [nul, 0, 1] este 1. Intervalul [nul, 0, 4] este 0.
  • Variaţia—Variaţia unui câmp numeric pe o pistă. Variaţia [1] este nulă. Variaţia [nul, 1,1, 1] este 1.
  • Deviaţia standard—Deviaţia standard a unui câmp numeric. Deviaţia standard [1] este nulă. Deviaţia standard [nul, 1,1, 1] este 1.

În câmpurile de tip şir puteţi calcula următoarele:

  • Calcul—Numărul şirurilor care nu sunt nule.
  • Oricare - Această statistică este un exemplu aleatoriu al unei valori de tip şir în câmpul specificat.
Toate statisticile sunt calculate pentru valori care nu sunt nule. Stratul tematic rezultat va conţine un câmp nou pentru fiecare statistică calculată. Se pot adăuga oricâte statistici alegând un atribut şi o statistică.

Selectare depozit de date

Rezultatele GeoAnalytics sunt stocate într-un depozit de date ArcGIS şi sunt expuse ca un strat tematic de obiecte spaţiale în Portal for ArcGIS. În cele mai multe situaţii, rezultatele trebuie stocate în depozitul de date spaţio-temporar, în mod implicit. În unele situaţii, salvarea rezultatelor în depozitul de date relaţionale este o opţiune bună. Următoarele sunt motivele pentru care aţi dori să stocaţi rezultatele într-un depozit de date relaţionale:

  • Utilizaţi rezultatele într-o colaborare portal la portal.
  • Activaţi capacităţile de sincronizare cu rezultatele dvs.

Nume strat tematic rezultat
